# Werewolf Game (Ma Sói)

Dự án game Ma Sói bao gồm cả Client (React Native/Expo) và Server (Node.js/Express/Socket.io).

## Cấu trúc dự án
- `werewolf-client/`: Mã nguồn ứng dụng di động (Sử dụng Expo).
- `werewolf-server/`: Mã nguồn máy chủ (Sử dụng Node.js và MongoDB).

## Hướng dẫn cài đặt và chạy

### 1. Server
```bash
cd werewolf-server
npm install
npm start
```
*Lưu ý: Bạn cần cài đặt MongoDB và cấu hình chuỗi kết nối (nếu có).*

### 2. Client
```bash
cd werewolf-client
npm install
npx expo start
```
*Dùng ứng dụng Expo Go trên điện thoại để quét mã QR hoặc chạy bản Web.*

## Tính năng chính
- Chơi game Ma Sói thời gian thực qua Socket.io.
- Đồ họa 2.5D/3D (Three.js/Fiber).
- Hệ thống nhân vật đa dạng.
